# CG Kupon Desktop Generator V2

Desktop app lokal untuk layout kupon PDF A3+ dengan cropmark.

## Fitur V2
- Auto Layout Engine V2
- Coba orientasi normal dan rotated
- Pilih layout dengan jumlah pcs/page paling banyak
- Contoh kupon 90x55 di A3+ 325x485 akan memilih 5x5 jika lebih optimal
- Build Windows EXE via GitHub Actions

## Build EXE via GitHub Actions
1. Buat repo GitHub baru.
2. Upload semua isi project ini ke root repo.
3. Masuk tab Actions.
4. Pilih workflow Build Windows EXE.
5. Klik Run workflow.
6. Download artifact CG-Kupon-Generator-V2-Windows.
7. Di dalamnya ada CG-Kupon-Generator-V2.exe.
